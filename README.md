## Secciones

- **Hero** — Presentación principal con animación de entrada
- **Stack** — Tecnologías y herramientas que uso
- **Experiencia** — Historial profesional y formación
- **Actividad** — Proyectos y actividad reciente
- **Proyectos** — Trabajos destacados con efecto parallax
- **Contacto** — Formulario y links de contacto

## Tecnologías

| Categoría    | Tecnologías                                         |
| ------------ | --------------------------------------------------- |
| Framework    | React 19, React Router DOM 7                        |
| Estilos      | Tailwind CSS 4, @midudev/tailwind-animations        |
| Iconos       | @tabler/icons-react, @radix-ui/react-icons          |
| Animaciones  | react-scroll-parallax, jarallax, @use-gesture/react |
| Herramientas | Vite 6, pnpm, PostCSS, Autoprefixer                 |

## Instalación

```bash
git clone https://github.com/Dylan10123/Portafolio-Dylan-Blasco.git
cd Portafolio-Dylan-Blasco
pnpm install
pnpm dev
```

## Scripts

| Comando        | Descripción                      |
| -------------- | -------------------------------- |
| `pnpm dev`     | Inicia el servidor de desarrollo |
| `pnpm build`   | Genera la build de producción    |
| `pnpm preview` | Previsualiza la build local      |
